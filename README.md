## Introduction

Provide some useful APIs for creating iOS auto layout constraints.

## Installation

EasyConstraint is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'EasyConstraint'
```
